How Invisible Text Works

Invisible text is created with Unicode characters that occupy space but are not displayed on screen. Popular invisible characters include:

  • U+2800 Braille Pattern Blank
  • U+3164 Hangul Filler
  • Zero Width Space (ZWSP)

Use Cases

  • Blank messages on WhatsApp, Instagram, or Facebook
  • Creating stylish usernames
  • Adding invisible spacing in text formatting
